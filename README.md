# CCSCalc
Cost of Goods Sold Calculator to increase profits and efficiency, and provide profitable sales strategies.
## Before start
Download this whole code and extract it anywhere you want. Or you can just type:  
```cmd
git clone https://github.com/MKhansa067/HPPCalc.git
cd HPPCalc
```
On your CMD!  
## After opening folder
Type:  
```cmd
npm install
npm i
```
## Run the project
Type:
```cmd
npm run dev
```
And acces through your browser:  
➜  Local:   http://localhost:8080/  
➜  Network: http://192.168.3.117:8080/  
➜  Network: http://172.23.96.1:8080/  
## Congratulations
You've ran the project!
