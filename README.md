# HPPCalc
<div align="justify">  
Kalkulator Harga Pokok Penjualan untuk meningkatkan laba dan efisiensi, serta memberikan strategi penjualan yang menguntungkan.  
</div>  

## Sebelum dimulai
<div align="justify">  
Unduh seluruh kode ini dan ekstrak di mana pun yang Anda inginkan. Atau bisa langsung ketik:  
</div>  

```cmd
git clone https://github.com/MKhansa067/HPPCalc.git
cd HPPCalc
```
Pada CMD Anda!  

## Setelah membuka folder
Ketik:  
```cmd
npm install
npm i
```

## Jalankan proyek ini
Ketik:
```cmd
npm run dev
```
Dan akses melalui Browser Anda:  
➜  Local:   http://localhost:8080/  
➜  Network: http://192.168.3.117:8080/  
➜  Network: http://172.23.96.1:8080/

Anda telah menjalankan proyeknya!
